# Soojin - Pet Sitter p2p Server

=====

## Reference

https://crosswalk-project.org/#documentation/webrtc

## Installation

  ```
  $ git clone https://github.com/petsitter/soojin.git
  $ cd soojin
  $ sudo npm install peer
  $ sudo npm install ip
  $ node index.js
  peer server running on 192.168.0.xxx:9000
  ```
